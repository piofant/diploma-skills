# diploma-skills

Три скилла для AI-агента (Claude Code / Codex / Cursor) под весь цикл диплома/ВКР: пройти антиплагиат и ИИ-детектор, оформить по требованиям вуза, подготовить защиту перед комиссией.

Откалибровано на реальной защите ИТМО (ФТМИ, 2026): первый прогон диплома — **25,52 % ИИ-контента**, после переписывания по чек-листу + autoresearch-петли против нейродетектора — **0 % ИИ**. Подробный разбор того, *как* — в посте [ссылка] и в `diploma-antiplag/REFERENCE.md`.

> ⚠️ Это не «генератор диплома» и не способ списать. Скиллы помогают убрать **ложные срабатывания** ИИ-детектора на тексте, который ты пишешь сам (детекторы палят «причёсанность», а не авторство), и привести работу к формальным требованиям. Ответственность за содержание — на тебе.

## Что внутри

| Скилл | Зачем |
|---|---|
| **`diploma-antiplag`** | Антиплагиат (оригинальность) + ИИ-детектор. 10-пунктный чек-лист признаков GPT-текста, 8 техник переписывания, postраничный риск-репорт. + autoresearch-петля: метрика «% ИИ» → переписать → замерить на детекторе → оставить если упало → повторять. |
| **`diploma-format`** | Оформление ВКР под требования вуза: заголовки, содержание, нумерация страниц и источников, дефисы, БД в приложении, академическая стилистика. + грабли build-pipeline (docx → Google Docs: синие заголовки, пустой TOC, нет нумерации). |
| **`diploma-defense`** | Презентация и защита перед ГЭК: структура 10–15 слайдов под 5–7 мин, принцип SHOW-not-TELL (реальные диаграммы вместо карточек), хронометраж речи, python-pptx тонкости, визуальная проверка рендером. |

Контекст калиброван на **ИТМО**, но 80 % применимо к любому вузу с системой Антиплагиат.ВУЗ + ИИ-детектором.

## Установка

### Claude Code (нативно — агент сам подхватит)

Каждый скилл — это папка с `SKILL.md`. Claude Code автоматически находит их в `~/.claude/skills/`.

```bash
git clone https://github.com/piofant/diploma-skills.git
cp -r diploma-skills/diploma-antiplag diploma-skills/diploma-format diploma-skills/diploma-defense ~/.claude/skills/
```

Или симлинком, чтобы тянуть обновления через `git pull`:

```bash
git clone https://github.com/piofant/diploma-skills.git ~/diploma-skills
ln -s ~/diploma-skills/diploma-antiplag ~/.claude/skills/diploma-antiplag
ln -s ~/diploma-skills/diploma-format   ~/.claude/skills/diploma-format
ln -s ~/diploma-skills/diploma-defense  ~/.claude/skills/diploma-defense
```

Проверка — спроси Claude: «какие скиллы про диплом у тебя есть». Дальше просто пиши задачу — агент сам решит, какой скилл тянуть:
- «проверь диплом на ИИ» / «доведи до 0% ИИ» → `diploma-antiplag`
- «оформи вкр по требованиям» / «содержание не bold» → `diploma-format`
- «сделай слайд защиты про X» / «собери презу под ГЭК» → `diploma-defense`

### Codex / Cursor / другие агенты (нет каталога скиллов)

У них нет авто-дискавери `~/.claude/skills/`, но `SKILL.md` — это просто инструкция в markdown. Два способа:

1. **Положить в проект и дать прочитать.** Склонируй репо рядом с работой и сошлись на нужный файл:
   - Codex: `@diploma-skills/diploma-antiplag/SKILL.md проверь по нему мой текст`
   - Cursor: добавь папку в контекст (`@diploma-antiplag`) и попроси применить.

2. **Вшить в правила агента.** Скопируй содержимое нужного `SKILL.md` (без YAML-шапки сверху) в:
   - Codex → `AGENTS.md` в корне проекта;
   - Cursor → `.cursorrules` или Project Rules;
   - любой чат-агент → системный промпт / первое сообщение.

Скиллы написаны как самодостаточные инструкции, специфичных для Claude Code инструментов почти нет — переносятся в любой агент.

## Autoresearch-движок (опционально)

«autoresearch-петля» из `diploma-antiplag` — это применение **стороннего** скилла **autoresearch** (MIT, © Kyle / drivelineresearch): автономный цикл «попробовал → замерил → оставил лучшее → повторил».

Сам цикл уже описан текстом внутри `diploma-antiplag/SKILL.md`, поэтому скилл работает и без него — петлю можно крутить руками. Но если хочешь полноценный движок экспериментов, ставь его отдельно:

👉 **github.com/drivelineresearch/autoresearch-claude-code**

Я его не перезаливаю сюда — это чужая работа, лучше брать апстрим (и обновления, и кредит автору).
